# Products

Crydon products are delivered as mission-ready **modules, reference designs, and integration kits** that accelerate program schedules. Availability and configuration depend on mission requirements and compliance constraints.

## Attritable UAS Reference Platform
A scalable UAV reference baseline intended for rapid iteration and payload experimentation.

**Typical uses**
- Training & evaluation
- ISR payload integration demonstrations
- Rapid mission prototyping

## Autonomy Stack (Integration Package)
A packaged approach to integrate and validate autonomy behaviors with common autopilot ecosystems and mission logic.

**Includes**
- Integration patterns and interfaces
- Safety and operational constraints
- Validation approach aligned with test events

## Datalink & Communications Integration Kit
Reference architectures and integration support for LOS/BLOS connectivity (subject to program constraints).

## Payload Integration Kit
Reusable mechanical/electrical/software patterns to integrate sensors and mission payloads quickly with reduced integration risk.
